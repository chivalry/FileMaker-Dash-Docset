Repeating functions
Repeating functions perform calculations on repeating fields.
Click a function name for details.
 
This function
Returns
Extend
In a calculation involving both repeating and non-repeating fields, allows a value in a non-repeating field to be used with every repetition in a repeating field.
GetRepetition
The contents of the specified repetition of a repeating field.
Last
The last valid, non-blank value in the specified field.
