Changing a tab control
You can add, rename, remove, and reorder tab panels in a tab control. You can also set the default front tab panel.
To change a tab control:
1.
In Layout mode, double-click the tab control.
Or, select a tab control and choose Format menu > Tab Control Setup.
The Tab Control Setup dialog box appears.
2.
Make changes to the tab control.
 
To
Do this
Add a new tab panel
For Tab Name, type a new name and click Create, or click Specify to create a tab label based on a calculation. See Specify Calculation dialog box.
Rename a tab panel
In the Tabs list, select a name, type the new name, and click Change, or click Specify to create a tab label based on a calculation.
Remove a tab panel
In the Tabs list, select a name and click Delete. All objects in the tab panel are deleted from the layout.
Reorder the tab panels
In the Tabs list, drag the double arrow next to the tab name.
Set the tab panel you want in front by default
For Default Front Tab, choose the panel you want in front when you switch from Layout mode to another mode.
When you switch to Layout mode from another mode, the front tab panel is the one that was last displayed in the previous mode.
3.
Click OK.
