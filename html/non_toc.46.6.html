Specify Label Layout (New Layout/Report assistant)
In this panel of the New Layout/Report assistant, you can choose a predefined measurement based on common label types, or specify a custom measurement.
To use predefined label dimensions that match the labels you have:
 •
From the Use label measurements for list, choose the label code that matches the labels you have.
FileMaker Pro creates a Labels layout that uses the exact dimensions of label stock code you choose. For example, choose Avery 5160 to have FileMaker Pro use that label’s measurements for the layout.
To create a custom label dimension:
1.
First, try to match your labels to a predefined dimension.
2.
If you can't match your labels to a predefined dimension, select Use custom measurements, and type values for Labels across the page, Width, and Height.
3.
Measure the unused margins on the label stock. Then, select Fixed page margins and enter values for Top, Bottom, Left, and Right.
