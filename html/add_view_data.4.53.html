Dates are not as expected
To avoid confusion when entering dates, always use four-digit years.
 •
Check the date formatting applied to the field. See Specifying formats for date fields.
 •
Make sure dates are entered into fields of type date. Choose File menu > Manage > Database, click the Fields tab, and check the field type of the field.
 •
Use field validation to prevent the wrong dates from being entered. For more information, see Defining field validation.
 •
Dates could have been entered before field validation options were added to the file.
