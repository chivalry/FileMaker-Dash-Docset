Edit Folder dialog box
Use this dialog box to change the name of a layout folder or a script folder.
To change the entire folder name, select the current name and type the new name. To change part of the folder name, select the portion to change, then type your changes. Then click OK.
The new name appears in the Manage Layouts or the Manage Scripts dialog box.
