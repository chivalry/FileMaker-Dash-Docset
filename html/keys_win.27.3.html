Mode keyboard shortcuts (Windows)
You can use keyboard shortcuts to change modes, navigate among records and fields, enter values in records and fields, and format and edit text.
 
To switch to
Press
Browse mode
Ctrl+B
Find mode
Ctrl+F
Layout mode
Ctrl+L
Preview mode
Ctrl+U

Object selection and modification keyboard shortcuts
 
To
Press
Select an object
Tab or Shift+Tab repeatedly until the desired object is selected.
Select multiple objects
Shift as you click each object individually.
Note  You can also drag the arrow pointer to make a selection box that includes the objects. The selection box does not have to completely surround the objects. (To select only the objects enclosed within the selection box, press Ctrl while dragging.)
Resize a selected object when the Inspector is open (click Inspector in the formatting bar)
Ctrl+1 to move to the Position tab of the Inspector. Then Tab to the desired sizing action and enter the size value you want. Press Enter on the numeric keypad to return to the Layout window.
Resize selected objects, keeping their proportions
Shift as you drag a handle.
Resize selected objects, keeping the difference in their lengths or widths the same.
Ctrl+Shift as you drag a handle.
The opposite edges of the objects remain fixed in the same position as you resize the objects.
Copy the style of a selected object
Ctrl+Alt+C
Paste the style of another object
Ctrl+Alt+V
