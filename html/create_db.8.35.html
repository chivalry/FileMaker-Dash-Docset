Changing field names
When you change a field name, FileMaker Pro changes the name on each layout and any field references used in calculation or summary fields. The field label changes on layouts if it matches the original field name.
To change a field name:
1.
With the database open, choose File menu > Manage > Database.
2.
In the Manage Database dialog box, click the Fields tab.
3.
If your database contains more than one table, select the appropriate table from the Table list.
4.
In the Fields tab of the Manage Database dialog box, select the field you want to rename.
5.
For Field Name, type or paste the new name.
6.
Click Change.
7.
Continue making changes or click OK.
To change a field name in Table View:
1.
With the database open in Browse mode, click Table View in the layout bar.
2.
Double-click the column heading for the field you want to rename, then type the new name.
