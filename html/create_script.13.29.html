Deleting a button
To delete a button:
1.
In Layout mode, select the button you want to delete.
2.
Choose Edit menu > Cut or Clear, or press Backspace or Delete.
When you delete a button, FileMaker Pro removes the button from the layout and deletes its definition.
To delete a button definition without deleting the button object:
1.
In Layout mode, select the button.
2.
Choose Format menu > Button Setup, or double-click the button.
3.
In the Button Setup dialog box, select Do Nothing, then click OK.
