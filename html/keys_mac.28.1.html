Keyboard shortcuts (OS X)
You can work with FileMaker Pro in four different modes: Browse, Find, Layout, and Preview. You can use keyboard shortcuts to invoke actions within modes and format and edit text.
Note  You can create your own keyboard equivalents if you use custom menus.
