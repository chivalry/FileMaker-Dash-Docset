Viewing extended privileges
The Manage Security dialog box displays a list of all the available extended privileges. You can view this list of extended privileges.
To view the extended privileges:
1.
Open the database file.
2.
Choose File menu > Manage > Security.
3.
In the Manage Security dialog box, click the Extended Privileges tab.
See Editing which privilege sets may use an extended privilege and Creating and deleting extended privileges.
