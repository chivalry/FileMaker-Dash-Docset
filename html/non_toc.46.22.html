How FileMaker Pro displays formatting attributes
When you change display attributes for objects, layout parts, or the layout background, FileMaker Pro displays formats according to a defined hierarchy, or precedence. One factor that determines whether format attributes display is whether you save changes at the layout level or the theme level.
