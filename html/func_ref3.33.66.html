KanjiNumeral
Purpose 
Converts Arabic numerals to Kanji numeral.
Format 
KanjiNumeral(text)
Parameters 
text - any text expression or text field
Data type returned 
text
Originated in 
FileMaker Pro 6.0 or earlier
Examples 
KanjiNumeral(123) returns
KanjiNumeral(““) returns
