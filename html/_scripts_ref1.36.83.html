Extend Found Set
Purpose 
Broadens the existing found set using the criteria in the stored find request.
Format 
Extend Found Set [Restore]
Options 
Select Specify find requests or click Specify to create and store a find request with the script step.
Compatibility 
 
Where the script step runs 
Supported 
FileMaker Pro 
Yes 
FileMaker Server scheduled script 
Yes 
FileMaker Go 
Yes 
Custom Web Publishing 
Yes 
FileMaker WebDirect 
Yes 
Runtime solution 
Yes 
Originated in 
FileMaker Pro 6.0 or earlier
Description 
This script step is equivalent to a logical OR search.
Example 1 
Extends the found set to include unpaid invoices.
Show Custom Dialog ["Include unpaid invoices in the found set?"]
If [Get ( LastMessageChoice ) = 1]
Extend Found Set [Restore]
End If
