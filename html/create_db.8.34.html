Changing field definitions
After you define a field, you can change it. To change the definitions of related fields, make the changes in the related table or file.
Important  Changing a field definition can change or delete data already stored in the field. For example, changing a field to a calculation field replaces existing data with a calculated value. Deleting a field definition erases any data stored in the field.
