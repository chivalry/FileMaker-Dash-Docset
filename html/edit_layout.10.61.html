Setting spell checking for individual fields
You can set FileMaker Pro to check spelling in two ways:
 •
For an entire file. For more information, see Setting file options.
 •
On a field-by-field basis.
To disable spell checking on a field-by-field basis:
1.
In Layout mode, select a field. Or, to specify attributes for fields you add later, start with no fields selected.
2.
Click Inspector in the layout bar, then click Data.
3.
In the Behavior area, select Do not apply visual spell-checking.
