Ceiling
Purpose 
Returns number rounded up to the next integer.
Format 
Ceiling(number)
Parameters 
number - any numeric expression or field containing a numeric expression
Data type returned 
number
Originated in 
FileMaker Pro 7.0
Examples 
Ceiling(1.25) returns 2.
Ceiling(-1.25) returns -1.
