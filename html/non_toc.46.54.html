Using the chart preview
You can use the chart preview to work efficiently in the Chart Setup dialog box.
Note  In a shared file, data changes made by other users are not displayed in the chart preview. However, the chart will display updated data when it is viewed in Browse mode.
 
To
Do this
Display tooltips for objects in the preview
Pause the pointer over the title, chart legend, axes titles, axes, or radius objects.
Tip  You can select an object in the preview to open the Chart inspector and activate the object’s control.
Switch the x-axis and y-axis
Click Swap Axes. If the chart has multiple y data series defined, only the first y-series appears on the x-axis.
Display sample data if the preview is slow to display
Click Use actual data below the preview and choose Use sample data. When you’ve finished editing the chart, restore the Use actual data setting or verify the chart in Browse mode.
Tip  You can choose View menu > Show > Sample Data in Layout mode to display sample data in the Chart Setup dialog box by default.
