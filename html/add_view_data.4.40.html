Playing QuickTime and QuickTimeVR movies in container fields
You can play QuickTime movies and view QuickTime VR movies that are stored in container fields. For more information about QuickTime files in container fields, see Using data in container fields.
Note  Windows: If you’re resizing a window in which a QuickTime movie is playing, the movie briefly stops displaying until you’ve finished resizing the window.
To play a QuickTime movie:
In Browse mode, click the container field that contains the movie. The following table describes QuickTime controls.
