Specifying a menu set for a layout
For each layout, you can specify which menu set displays when a layout is active. A menu set is the collection of menus that installs on the menu bar.
To specify a menu set for a layout:
1.
In Layout mode, choose the layout you want to work with from the Layout pop-up menu.
2.
Click Layout Setup in the layout bar.
You can also click the name of the layout’s current table.
3.
In the Layout Setup dialog box, select a menu set from the Menu Set list.
4.
Click OK.
Notes
 •
Choosing [File Default] maintains the default menu set specified for the file. To override the default menu set for this layout, choose a different menu set from the list.
 •
If you have FileMaker Pro Advanced, you can choose Manage Custom Menus from the Menu Set list to open the Manage Custom Menus dialog box. See Defining custom menus (FileMaker Pro Advanced) or the FileMaker Pro Advanced Features Guide.
