Save as Layout dialog box
If you have the correct privileges, you can save a quick chart in a new layout. You can re-display or edit the chart later.
Note  You can’t save a quick chart on an existing layout.

Click or select
Print the chart
Print Chart
Save the chart in a new layout
Save as Layout
To access the chart later via the Layout pop-up menu
Include in layout menus
If you clear Include in layout menus, you can use Layouts menu > Go To Layout or the Manage Layouts dialog box to display the chart in layout mode. See Managing layouts.
