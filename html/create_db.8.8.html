About defining database fields
A field is the basic unit of data entry in a record. To define a new field, you give it a name. Then you select options that determine how the field interprets, enters, calculates, stores, and displays data.
After you define a field, you can set validation, auto entry, and storage options. See Setting options for fields.
There are three ways to define fields in a database:
 •
In Layout mode using the Field Picker dialog box (see Defining fields in the Field Picker dialog box)
 •
In the Manage Database dialog box (see Defining fields in the Manage Database dialog box)
 •
In Table View (see Defining fields in Table View)
 
Note  To add an existing field to a layout, go to Layout mode and drag a field from the Field tool . See Placing and removing fields on a layout.
