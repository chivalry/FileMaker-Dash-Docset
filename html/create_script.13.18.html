Apple events error: "Invalid object containment" (-1723)
This error alerts you that an object may not have been specified correctly according to the FileMaker Pro object containment hierarchy. The object hierarchy tells you how FileMaker Pro objects are related. For instance, fields are a subset of records but not menus; menus can belong only to an application, not to a window or database.
To correct this error: Examine the relationship of the objects you have specified and compare them with the containment hierarchy.
The containment hierarchy can be found in the FileMaker Pro event and object dictionary accessible with the AppleScript Script Editor.
