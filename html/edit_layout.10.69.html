Rotating objects
You can rotate fields, text blocks, and graphic objects in 90-degree increments.
To rotate one or more objects:
1.
In Layout mode, select the objects that you want to rotate.
For more information, see Selecting objects.
2.
Choose Arrange menu > Rotate.
The object or objects rotate 90 degrees clockwise. To rotate the object or objects further, choose Arrange menu > Rotate again.
Notes
 •
When you click or tab into a rotated field in Browse mode, the field is temporarily displayed unrotated while you edit the text in the field.
 •
You cannot rotate portals, tab controls, slide controls, web viewers, or popovers; however you can rotate popover buttons.
