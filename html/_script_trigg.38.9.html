OnRecordLoad
Triggers a script to run after a record is made current or entered, such as after a user or script switches to a layout, switches records, opens a new window, creates or deletes a record, or performs a Find operation.
When the script runs
After the event has been processed.
Modes in which the trigger can be used
Browse and Find modes.
Result
None.
Originated in 
FileMaker Pro 10.0
Notes
 •
FileMaker Pro does not pass information regarding which record activated the script. You may want to use additional script triggers to capture that information.
 •
See Changing object data without activating script triggers for a list of actions that do not activate this script trigger.
