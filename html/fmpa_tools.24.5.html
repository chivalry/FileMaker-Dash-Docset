Encrypting database files (FileMaker Pro Advanced)
You can encrypt database files by using the Database Encryption feature of FileMaker Pro Advanced. Encryption protects FileMaker database files from unauthorized access while the files are being stored on disk. Temporary files that are created by encrypted files are also encrypted.
You create an encryption password for the file, which protects the data if the file is copied or stolen. Users who do not enter the encryption password are not allowed access to the file.
Encrypted files can be decrypted as needed.
Note  Encrypting database files that are part of a runtime solution is not supported.
