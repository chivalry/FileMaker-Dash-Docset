Working with popovers on layouts
Use popovers to simplify entering and viewing data. A popover is a layout object that allows you to work with fields, text, and other objects on a layout. Popovers allow you to access additional details or controls for your data without having to move to another layout or window, helping you to maintain the context of your work.
To add a popover to a layout, in Layout mode create a popover button. An associated popover is created along with the popover button. You can then add fields and other objects to the popover.
To work with a popover in Browse mode, you click the popover button to open the popover. You can enter and view data in fields on a popover, view charts, use web viewers, work with panel controls, and so on, just as you do in a layout.
