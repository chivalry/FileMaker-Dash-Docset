Adding filename extensions to files (FileMaker Pro Advanced)
You can use the Developer Utilities to give non-runtime FileMaker database files the FileMaker file extension. This is useful for cross-platform development, if you want the filename extension to be consistent on both Windows and Macintosh platforms.
To add the FileMaker filename extension to non-runtime databases:
1.
Close all of your database files that you are going to customize.
2.
Choose Tools menu > Developer Utilities.
3.
If you have used the Developer Utilities on the same database before and saved your settings, click Load Settings.
4.
Click Add to locate the files that you want to customize.
5.
If you add multiple files, double-click a file in the list to specify the primary file.
6.
Select Developer Utilities settings as required. See Setting up files for developer solutions (FileMaker Pro Advanced).
7.
For Solution Options, click Specify. See Specifying options for developer solutions (FileMaker Pro Advanced).
8.
Select Databases must have a FileMaker file extension, and click OK.
