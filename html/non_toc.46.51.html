Authenticate Script Debugging dialog box
This dialog box appears if you do not have Full Access privileges when you click to authenticate a script in the Data Viewer or Script Debugger.
To authenticate a script:
1.
Enter a user name and password with Full Access privileges.
2.
Click OK.
