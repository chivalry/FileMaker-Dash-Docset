Specify Excel Data dialog box
Use the Specify Excel Data dialog box when you’re importing data from a Microsoft Excel file that contains more than one worksheet or includes a named range.
 •
To import data by worksheet, select Display worksheets, select the worksheet you want to import, then click Continue.
 •
To import data by named range, select Display named ranges, select the named range you want to import, then click Continue.
