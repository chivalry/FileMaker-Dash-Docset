Working with plug-ins
C or C++ programmers familiar with calculations in FileMaker Pro and FileMaker Pro Advanced can create plug-ins that extend the feature set of FileMaker Pro.
This section describes how to:
 •
Create plug-ins and make them available to users.
 •
Install plug-ins in solution files.
 •
Set up solution files to check plug-in version information and update plug-ins automatically, if needed.
