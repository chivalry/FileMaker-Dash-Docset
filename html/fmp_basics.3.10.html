Using shortcut menus
You can quickly edit an object or data by choosing commands directly from a shortcut menu, also called a “context menu” or “contextual menu.”
To display a shortcut menu:
Right-click or Control-click an object or an area to see a list of commands.
Notes
 •
Shortcut menu commands vary based on what mode you’re in, the item the cursor is over, and whether an item is selected. For example, when a field is selected in Layout mode, you have access to commands that pertain to Layout mode.
 •
You can use the Control Panel (Windows) or System Preferences (OS X) to configure your mouse, trackpad, or input device. For example, you can set up a secondary button on a mouse.
