Working with favorite files and hosts
You can add a FileMaker Pro file located on your computer or on a remote computer to your list of favorites in the File menu and in the Quick Start Screen. You can also add a remote host to the list of favorites.
To add the current file to your list of favorites, choose File menu > Open Favorite > Add Current File to Favorites.
To manage favorite files and hosts:
1.
Choose File menu > Open Favorite > Manage Favorites. Or, in the FileMaker Pro Quick Start Screen, click Manage Favorites.
The Manage Favorites dialog box appears.
 
To
Do this
Add a favorite local file
Click to open the Add a Favorite dialog box (Windows) or Open dialog box (OS X), select one or more files in the list, click Select, then close the Manage Favorites dialog box.
Add a favorite remote file
Click to open the Add a Favorite dialog box (Windows) or Open dialog box (OS X). Click Remote. Select the host on which the file resides, select a remote file, then click Add to Favorites. Click Close to return to the Add a Favorite dialog box (Windows) or Manage Favorites dialog box (OS X). Click Cancel to return to the Manage Favorites dialog box (Windows).
Tip  When you double-click a collapsed host name or IP address, the list of remote files is refreshed.
Change the order of favorite files
Select a file and drag it to a new location in the list.
Open a file
Double-click the file in the list.
Add a favorite remote host
Click to open the Add a Favorite dialog box (Windows) or Open dialog box (OS X). Click Remote. Select the host, then click Add to Favorite Hosts. Click Save in the Edit Favorite Host dialog box. Click Close to return to the Add a Favorite dialog box (Windows) or Manage Favorites dialog box (OS X). Click Cancel to return to the Manage Favorites dialog box (Windows).
Remove a file or host from the favorites list
Select a file or host name and click .
Change the remote host settings
Select a host, click . Choose Edit Host. For information about changing the settings for the remote host, see Opening shared files as a client.
Update the list of expanded hosts
Click . Choose Refresh Expanded Hosts.
2.
Close the Manage Favorites dialog box.
