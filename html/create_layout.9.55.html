Controlling how layouts print
What printer you use, what printer options you choose, the margins you specify, and different layout options you choose all affect how FileMaker Pro prints your data.
FileMaker Pro also allows you to set up a layout to print records in columns (for example, to print a multi-column phone directory), set sliding options to reduce blank spaces in printouts, and prevent certain layout objects from printing.
