Select ODBC Data Source dialog box
This dialog box appears so that you can access an existing ODBC data source. You can import from the data source, or you can work with the data source interactively in the relationships graph.
 •
If you want to import ODBC data into FileMaker Pro files (by writing SQL queries), see Querying an ODBC data source from FileMaker Pro.
 •
If you want to access and work with ODBC data interactively on layouts and in the relationships graph (without writing SQL queries), see the section on adding an ODBC data source in Connecting to data sources.
